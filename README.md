# silktide-consent-manager-wp-plugin
Lägger till Silktide Consent Manager v2.0 som plugin i WordPress. Testad med WordPress 6.9.1.

## Beskrivning
Detta är i stort sett https://github.com/silktide/consent-manager anpassad för att kunna laddas upp som plugin i WordPress. Språket är svenska men kan översättas genom att redigera texterna i silktide-consent.php. Innehållet i Assets-mappen kan ersättas av filerna som ingår i _Download Cookie Banner Files_ när man laddar ner från https://silktide.com/consent-manager/

## Installation
Ladda ner filerna, packa innehållet som .zip och lägg till i WordPress via Tillägg-menyn. Ladda inte upp zip-filen direkt från github eftersom den innehåller huvudmapp -> mapp.
